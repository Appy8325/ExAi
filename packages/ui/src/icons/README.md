# icons

Custom Lucide-style icon components for concepts Lucide's catalog lacks
(booth, badge-scan, floor-plan, lead-pipeline). Drawn in-house on a 24px
grid, 2px stroke, matching Lucide's optical style, and shipped as React
components with the same size/stroke props as `lucide-react`
(docs/39-design-system.md §14.2).

Icon color inherits `currentColor` — components here consume text tokens,
never palette primitives.

New custom icons require design review; the bar is "Lucide genuinely has no
fit," checked against the Lucide catalog first.

No custom icons are required for Milestone 0. Populated as needed.
