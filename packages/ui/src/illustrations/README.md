# illustrations

Spot illustrations (empty states, onboarding, upgrade prompts) built only
from semantic tokens, so they theme automatically across light/dark and
event-accent contexts. Shipped as React SVG components
(docs/39-design-system.md §14.3).

Max 3 hues per illustration: neutral structure + brand accent + one
status/ai accent when meaningful. No external illustration libraries, no
raster illustrations, no photography.

No illustrations are required for Milestone 0. Populated as needed.
