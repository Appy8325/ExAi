# Design System

## Direction

ExAi now uses a premium, calm SaaS visual language inspired by Linear, Stripe, Vercel, Notion, Arc Browser, and Apple software.

## Principles

- **Clarity first:** every surface should explain the user's next best action.
- **Depth with restraint:** use layered cards, subtle gradients, and soft shadows instead of noisy decoration.
- **Role-aware hierarchy:** organizer, exhibitor, attendee, and admin screens should feel connected but purpose-built.
- **Accessible motion:** animation should clarify state changes and must respect reduced-motion preferences.

## Tokens and utilities

- `shadow-premium` provides a consistent elevated surface treatment for hero panels, demo cards, and featured modules.
- `focus-premium` provides a high-contrast keyboard focus treatment for custom interactive elements.
- `animate-fade-up` is available for tasteful entrance transitions when layout stability is guaranteed.

## Component guidance

- Use rounded cards with clear headings, concise supporting copy, and one obvious primary action.
- Prefer customer-facing language over implementation terminology.
- Empty states should explain what is missing, why it matters, and what the user can do next.
